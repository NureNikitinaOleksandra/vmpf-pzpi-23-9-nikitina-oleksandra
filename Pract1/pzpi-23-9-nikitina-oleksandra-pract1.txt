4. Напишіть функцію, яка приймає три параметри (a, b, c) і виводить на екран найменше з них.

	Хід вирішення:
Я розділила програму на функції, щоб кожна виконувала одну задачу:
• reading() – для введення даних;
• checking() – для перевірки правильності введення;
• find_min(a, b, c) – для знаходження найменшого числа;
• main() – головна функція.
def reading():
    parameters = input("Введіть три числа через пробіл і натисніть Enter: ")
    parameters = parameters.split()
    return parameters

def checking(parameters):
    if len(parameters) != 3:
        print("Потрібно ввести рівно три значення.")
        return False

    for parameter in parameters:
        try:
            float(parameter)
        except ValueError:
            print(f"Введене значення {parameter} не є числом.")
            return False

    return True

def find_min(a, b, c):
    if a < b:
        if a < c:
            return a
        else:
            return c
    else:
        if b < c:
            return b
        else:
            return c

def main():
    while True:
        parameters = reading()

        if checking(parameters):
            a, b, c = float(parameters[0]), float(parameters[1]), float(parameters[2])

            result = find_min(a, b, c)

            print("Найменше число:", result)
            break

if __name__ == "__main__":
    main()

def reverse_string(string):
    reversed_string = ""
    i = len(string) - 1

    while i >= 0:
        reversed_string += string[i]
        i -= 1

    return reversed_string

4. Реалізуйте програму, яка визначає, чи є слово паліндромом (читається однаково з обох боків).

	Хід вирішення:
Для вирішення цього завдання я взяла функцію reverse_string(string) з минулого завдання та створила функцію is_palindrome(word), де початкове слово порівнюється з перевернутим.
def reverse_string(string):
    reversed_string = ""
    i = len(string) - 1

    while i >= 0:
        reversed_string += string[i]
        i -= 1

    return reversed_string

def is_palindrome(word):
    reversed_word = reverse_string(word)
    return word.lower() == reversed_word.lower()

text = input("Введіть щось: ")

if is_palindrome(text):
    print(f"Слово {text} є паліндромом")
else:
    print(f"Слово {text} не є паліндромом")

4. Розробіть алгоритм сортування масиву чисел методом швидкого сортування (QuickSort) та виведіть відсортований масив.

	Хід вирішення:
Я реалізувала алгоритм швидкого сортування quick_sort(a). Його зміст такий:
• якщо масив містить один або нуль елементів, він уже відсортований;
• обирається опорний елемент (х) випадковим чином;
• усі елементи масиву розподіляються на три списки:
1) low – елементи, менші за х;
2) equal – елементи, рівні х;
3) high – елементи, більші за х;
• для списків low та high рекурсивно викликається функція quick_sort;
• після цього відсортовані частини об'єднуються у новий список.
Також у цій функції реалізована рекурсія.
import random

def quick_sort(a):
    if len(a) <= 1:
        return a
    
    x = a[random.randint(0, (len(a) - 1))]
    
    low = []
    equal = []
    high = []

    for y in a:
        if y < x:
            low.append(y)
        elif y == x:
            equal.append(y)
        else:
            high.append(y)

        a = quick_sort(low) + equal + quick_sort(high)

    return a

array = [2, 4, 12, 7, 23, 5, 2, 89]

sorted_array = quick_sort(array)

print(array)
print(sorted_array)
